
# PPTX File Renamer
Python automation script to batch rename PPTX files based on structured information extracted from the first slide.

## Overview
This project automates a repetitive file management task by parsing text data from PowerPoint files and applying a standardized naming convention.

It demonstrates skills in:
- Data extraction from semi-structured files
- Text processing with regular expressions
- File system automation using Python

## How it works
- Reads the first slide of each PPTX file
- Extracts year, protocol number, optional CODE flag and title
- Renames files following a consistent pattern

## Tech stack
- Python
- python-pptx
- pathlib
- regex

## Usage
1. Set the target folder path in the script
2. Install dependencies
3. Run the script
